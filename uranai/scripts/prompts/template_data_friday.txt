## ルール（厳守）

### intro_text
- 「今日いいことが起こりそうな生まれ年のTOP10を発表します。あなたの生まれ年、入ってるかな？」風の挨拶
- 月相を「〜の頃」「〜かも」など柔らかく織り込む（断定NG）
- 金星「金曜は金星の支配日」を1回さりげなく
- 40〜80字
- 「おはようございます、豊川ガイドです🦊」は別途出力されるので含めない
- 「おはよう」「こんにちは」など挨拶語で intro_text を始めないこと（直前に挨拶があるため重複する）
- 「※」「明日も」等の固定文言は含めない

### items（必ず10件・あなたが順位を決定）
- 入力 years 配列の10年を、ランキング順（1位→10位）に並べ替える
- 全年ポジティブ基調（全部ラッキー）・stars は使わない
- ラベル形式：「YYYY年（元号N年）生まれ」（必ず和暦併記）
- comment は **必ず2文構成（句点で2文）**・各文 **18文字以内（厳守！）**・合計36文字以内
- リール動画とアイキャッチのカード内に収めるため、長文厳禁。1文目で運勢の本質・2文目で具体的なアドバイス
- 例（理想）：「縁起のいい1日です。新しい挑戦を始めてみて。」(各15字・合計28字)
- 18字を超える文は **絶対にNG**（カードからはみ出して見栄えが崩れる）
- 1〜2か所だけラッキースポットへの言及を自然に織り込む（全年には入れない）
- ラッキースポット名はそのまま【】なしで使用
- 専門用語の解説や押し付けがましさNG
- 標準語・砕けた口調・関西弁NG

### 元号換算ルール（絶対厳守）
通常年：「YYYY年（元号N年）生まれ」
- 例：1990年（平成2年）生まれ
- 例：1976年（昭和51年）生まれ
- 例：2020年（令和2年）生まれ

境界年（2年のみ）：両元号を併記
- 1989年（昭和64年/平成元年）生まれ
- 2019年（平成31年/令和元年）生まれ

#### 元号換算表
- 昭和：1926年〜1989年（昭和64年/1月7日まで）
- 平成：1989年（平成元年/1月8日）〜2019年（平成31年/4月30日）
- 令和：2019年（令和元年/5月1日）〜

具体的な計算：
- 西暦 - 1925 = 昭和年（1926〜1988年）
- 西暦 - 1988 = 平成年（1990〜2018年）
- 西暦 - 2018 = 令和年（2020年〜）

### ラッキースポット名の絡め方
- is_chain == true → 「お近くの〇〇」
- is_chain == false → 「〇〇に立ち寄ってみては？」など
- 共通フレーズ：「立ち寄ってみては？」「覗いてみて」「行ってみて」「ぶらっと」「足を運んで」から自然に選ぶ
