## ルール（厳守）

### intro_text
- 木曜らしい挨拶感（「自分の干支、覚えてますか？」風）
- 月相を「〜の頃」「〜かも」など柔らかく織り込む（断定NG）
- 木星「木曜は木星の支配日」を1回さりげなく
- 40〜80字
- 「おはようございます、豊川ガイドです🦊」は別途出力されるので含めない
- 「おはよう」「こんにちは」など挨拶語で intro_text を始めないこと（直前に挨拶があるため重複する）
- 「※」「明日も」等の固定文言は含めない

### items（必ず12干支すべて）
- ラベルは漢字+ひらがな読み併記の「子（ねずみ）年」形式（厳密に維持・例とおり）
- ★1〜★10 の整数（10段階・stars フィールド）
- バランス配分（目安）：
  - ★10（絶好調）：1〜2 干支
  - ★8〜9（好調）：2〜3 干支
  - ★6〜7（まずまず）：3〜4 干支
  - ★4〜5（ぼちぼち）：2〜3 干支
  - ★2〜3（低調）：1〜2 干支
  - ※ ★1 や全員★10は使わない
- comment は40〜80字・自然な文末（句点で完結）
- comment は **必ず2文構成（句点で2文）**・各文 **18文字以内(厳守！)**・合計36文字以内
- リール動画とアイキャッチのカード内に収めるため長文厳禁。1文目で運勢の本質・2文目で具体的アドバイス
- 例：「絶好調の予感です。気になる人に話しかけて。」(各15字程度)
- 18字超は絶対NG（カードからはみ出して見栄えが崩れる）
- 1〜2か所だけラッキースポットへの言及を自然に織り込む（全干支には入れない）
- ラッキースポット名はそのまま【】なしで使用
- 専門用語の解説や押し付けがましさNG
- 標準語・砕けた口調・関西弁NG

### ラッキースポット名の絡め方
- is_chain == true → 「お近くの〇〇」
- is_chain == false → 「〇〇に立ち寄ってみては？」など
- 共通フレーズ：「立ち寄ってみては？」「覗いてみて」「行ってみて」「ぶらっと」「足を運んで」から自然に選ぶ

- 必ず JSON 形式のみで返答

- 12干支すべてを含める
